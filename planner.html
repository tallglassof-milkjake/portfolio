Scheduler

A Day Planner

One of my assignments from the Full Stack Coding Bootcamp was to practice using jQuery, moment.js, DOM manipulation and the use of local storage and create a simple day scheduler.
To understand the basics of moment.js I used DOM manipulation to take in the local current time and display each hour block in a colour depending on the time. Green for upcoming, Blue for current and Grey to display the previous hours.


For the local storage aspect of this task I used JSON parse and JSON stringify to push and pull data from local storage and then a little DOM manipulation to throw it into corresponding time blocks for that day.


This application has some updates coming. More presentable UI, better use of local storage, the ability to plan ahead much like with a calendar and more!
