SunnySideUp Weather

Forecaster

Another project from Full Stack Coding Bootcamp! This time I was tasked with using and navigating API calls using AJAX. Specifically the API from the openweathermap.org website.


I started this assignment by researching and reading documentation provided by the openweather team. This gave me the confidence needed to tackle the API calls and understand the way AJAX works. Then I created a basic layout in HTML and CSS to start, and got to tackling some jQuery functions to grab some info from openweather and start bringing data to life on the page.


The main section of this application displays current weather for the city the user enters. The openweather API handles the temperature in kelvin giving me the opportunity to play with calculations to interpret into celsius, with future plans to display in farenheit when in countries that use that particular format. Other data displayed include humidity, temperature 'feels like' and wind speeds. It also shows a simple UV indicator dictated by a separate UV index API and some simple CSS and jQuery class manipulation.



Next up was bringing in the five day forecast. Yet another API was needed from openweather to fullfil this. It required calculations from the previous kelvin to celsius converter explained earlier. Then it was the case of selecting which time of the day the forecast data was drawn from and sending to the browser to display.


Finally, the search bar that is nested in the side bar section was given the ability to save searched data to local storage for the ease of the user. This would see the cities searched display as a list of buttons that can be called upon to assess the weather in those sections incase the user navigates away from the page or closes the browser.


There are some future modifications coming such as once the page has loaded it will immediately display the current weather and forecast for the users location if they have it turned on. Other modifications will include a more presentable UI, better functionality for local storage data and the ability to select more weather options including viewing a rain map.
